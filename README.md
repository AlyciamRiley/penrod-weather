
## Penrod Weather
 This application used the OpenWeather API to display the current weather conditions for the Penrod Offices located in Minneapolis, Milwaukee, Dallas, and Chicago. 

## Technologies Used
- Mithril.js
- Lightning Design System
- Node.js
- Webpack
- Express
